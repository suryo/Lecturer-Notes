# Kontrak Perkuliahan: Pemrograman Web 2 (Framework Laravel)

## 1. Identitas Mata Kuliah

- **Nama Mata Kuliah**: Pemrograman Web 2
- **Prasyarat**: Pemrograman Web 1 (HTML, CSS, JS, PHP Native)
- **Fokus**: Web Application Development using Laravel Framework

## 2. Deskripsi Mata Kuliah

Mata kuliah ini dirancang untuk memperkenalkan mahasiswa pada pengembangan aplikasi web tingkat lanjut menggunakan framework **Laravel**. Mahasiswa akan bertransisi dari penulisan kode PHP prosedural (Native) menuju arsitektur **Model-View-Controller (MVC)** yang modern, efisien, dan memiliki standar industri.

## 3. Capaian Pembelajaran (CPMK)

1. Mahasiswa mampu memahami konsep Framework Web dan arsitektur MVC.
2. Mahasiswa mampu mengelola database menggunakan Migration dan Eloquent ORM.
3. Mahasiswa mampu membangun sistem otentikasi dan otorisasi yang aman.
4. Mahasiswa mampu membangun RESTful API sederhana.
5. Mahasiswa mampu mendeploy aplikasi web Laravel ke lingkungan produksi.

## 4. Materi Pembelajaran (14 Pertemuan)

1. **Pengenalan PHP & Web Server**: Syntax Dasar, Variabel, dan Struktur Kontrol.
2. **Koneksi PHP ke MySQL**: CRUD Native (Create, Read, Update, Delete) & Prepared Statements.
3. **Intro to Framework & Laravel 11 Setup**: Arsitektur Modern & Instalasi.
4. **MVC Architecture, Routing, & Views**
5. **Blade Templating & Assets (Vite)**
6. **Database Migrations & Seeders**
7. **Eloquent ORM Basics**
8. **UTS / Midterm Project**
9. **Controllers & Form Validation**
10. **Eloquent Relationships (1:1 & 1:N)**
11. **Auth & Security (Starter Kits)**
12. **Middleware & File Uploads**
13. **Laravel Sanctum (APIs)**
14. **Deployment & Final Project Review**

## 5. Komponen Penilaian

- **Kehadiran**: 10%
- **Tugas Mandiri/Praktikum**: 30%
- **UTS (Project Progress)**: 30%
- **UAS (Final Web Project)**: 30%

## 6. Aturan Kelas

- Toleransi keterlambatan 15 menit.
- Tugas dikumpulkan secara tepat waktu melalui platform yang ditentukan.
- Plagiarisme kode akan berakibat pada pembatalan nilai tugas.
