# Tugas 3: Laravel Database & Eloquent

## Deskripsi Tugas

Membangun modul manajemen data menggunakan fitur Database Migrations, Seeders, dan Eloquent ORM.

## Kriteria Penilaian

1.  **Migrations**: Struktur tabel yang didefinisikan secara terprogram.
2.  **Seeders**: Data awal yang diinput secara otomatis.
3.  **Eloquent CRUD**: Interaksi data menggunakan class Model (Create, Read, Update, Delete).
4.  **Validation**: Implementasi aturan validasi pada form input (required, unique, dll).

## Instruksi Pengerjaan

1.  Buatlah tabel (misal: `products`) beserta migrasinya.
2.  Implementasikan fitur untuk menambah, mengedit, dan menghapus data produk tersebut.
3.  Pastikan form input memiliki validasi agar data di database tetap konsisten.
