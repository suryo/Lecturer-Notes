# Panduan UAS: Advanced Web Application Ecosystem

## Deskripsi

Membangun aplikasi web utuh yang mengintegrasikan fitur-fitur lanjutan Laravel seperti Keamanan, Manajemen File, dan Layanan API.

## Fitur Wajib

1.  **Authentication & Multi-Role**: Sistem login dengan hak akses berbeda (Admin vs User/Customer).
2.  **Multimedia Management**: Fitur upload dan manajemen file/gambar.
3.  **RESTful API**: Menyediakan endpoint API untuk konsumsi eksternal (menggunakan Sanctum).
4.  **Security Middleware**: Proteksi rute-rute sensitif secara tepat.

## Persyaratan Teknis

- Laravel 11.
- Database Relasional (MySQL).
- Frontend (Bootstrap/Tailwind).

## Rubrik Penilaian

- Implementasi Auth & Security: 30%
- Manajemen File & UI/UX: 25%
- API Development: 25%
- Kompleksitas Fitur & Struktur Kode: 20%
