# Tugas 2: Laravel Dasar (Routing & Blade)

## Deskripsi Tugas

Melakukan migrasi antarmuka statis ke dalam sistem templating Laravel. Tugas ini berfokus pada penguasaan Routing, Controller, dan Blade Templating.

## Kriteria Penilaian

1.  **Routing**: Definisi rute yang efisien di `web.php`.
2.  **Blade Inheritance**: Penggunaan `@extends`, `@section`, dan `@yield` secara tepat.
3.  **Blade Directives**: Implementasi logika tampilan menggunakan `@if` dan `@foreach`.
4.  **Controller Integration**: Logika pengiriman data dilakukan melalui Controller, bukan langsung di Route Closure.

## Instruksi Pengerjaan

1.  Buat project Laravel baru.
2.  Implementasikan sebuah layout utama (Master Layout).
3.  Buat minimal 3 halaman (Beranda, Profil, Layanan) yang mewarisi layout tersebut.
4.  Kirimkan data array dari Controller untuk ditampilkan di halaman menggunakan `@foreach`.
