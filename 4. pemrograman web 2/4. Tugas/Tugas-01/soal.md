# Tugas 1: Dasar PHP & MySQL Native

## Deskripsi Tugas

Membangun aplikasi web sederhana menggunakan PHP Native (tanpa framework) yang mampu melakukan manipulasi data (CRUD) pada satu tabel database. Tugas ini bertujuan untuk memastikan pemahaman mendalam tentang siklus request-response dan interaksi basis data secara manual.

## Kriteria Penilaian

1.  **Koneksi Database**: Menggunakan `mysqli` atau `PDO` untuk terhubung ke MySQL.
2.  **Operasi CRUD**:
    - **Create**: Mampu menyimpan data dari Form ke Database.
    - **Read**: Menampilkan data dalam tabel HTML.
    - **Update**: Mengubah data yang sudah ada.
    - **Delete**: Menghapus data dari database.
3.  **Algoritma & Logic**: Penggunaan variabel, array, dan struktur kontrol yang tepat.
4.  **Keamanan Dasar**: Implementasi sanitasi input sederhana.

## Instruksi Pengerjaan

1.  Siapkan database bernama `tugas1_pweb2`.
2.  Buat tabel `mahasiswa` atau `buku` dengan minimal 5 kolom.
3.  Implementasikan seluruh fungsi CRUD dalam beberapa file PHP terpisah atau satu file yang terorganisir.
4.  Gunakan CSS sederhana agar tampilan tabel dan form nyaman dilihat.
