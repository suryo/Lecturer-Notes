# Pertemuan 13: RESTful API Development (Sanctum)

## Deskripsi Singkat

Membangun API menggunakan Laravel dan mengamankannya dengan **Laravel Sanctum**.

## Materi Pembelajaran

- **API Routing**: `routes/api.php`.
- **JSON Response**: Format data untuk mobile/front-end.
- **API Tokens**: Otentikasi stateless via Sanctum.

## Praktikum

1. Buat API endpoint untuk data kategori.
2. Uji coba request menggunakan Postman.
