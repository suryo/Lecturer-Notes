# Pertemuan 7: Eloquent ORM Basics (CRUD)

## Deskripsi Singkat

Mempelajari fitur **Eloquent ORM** untuk berinteraksi dengan database menggunakan objek PHP tanpa query SQL manual.

## Materi Pembelajaran

- **Model**: Representasi tabel.
- **CRUD**: `all()`, `find()`, `create()`, `update()`, `delete()`.
- **Mass Assignment**: Variabel `$fillable`.

## Praktikum

1. Buat Model `Category`.
2. Eksperimen CRUD via **Tinker**.
