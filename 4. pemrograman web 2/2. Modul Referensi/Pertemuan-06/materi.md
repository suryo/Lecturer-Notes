# Pertemuan 6: Database Migrations & Seeders

## Deskripsi Singkat

Mempelajari cara mengelola skema database secara terprogram menggunakan **Migrations** dan mengisi data dummy menggunakan **Seeders**.

## Materi Pembelajaran

- **Migration**: Version Control untuk database (`php artisan migrate`).
- **Seeder**: Pengisian data otomatis (`php artisan db:seed`).

## Praktikum

1. Buat migration untuk tabel `categories`.
2. Buat seeder untuk mengisi data kategori awal.
