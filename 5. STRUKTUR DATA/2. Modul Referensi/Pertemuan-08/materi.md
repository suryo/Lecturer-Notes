# Pertemuan 8: Midterm Break & Materi Review (Struktur Data)

## Deskripsi Singkat

Sesi ini meninjau kembali konsep-konsep dasar struktur data dan algoritma yang telah dipelajari dari Pertemuan 1 hingga 7. Mahasiswa fokus pada pemahaman logika implementasi menggunakan bahasa C++ serta analisis kompleksitas waktu.

## Topik Utama yang Direview

1. **Dasar C++**: Manipulasi Pointer dan Alokasi Memori Dinamis (`new`/`delete`).
2. **Array**: Operasi pada array 1D dan 2D (Matrix).
3. **Linked List**: Logika penyambungan node (Next/Prev) pada Single dan Double Linked List.
4. **Linear Data Structure**: Perbedaan mekanisme LIFO (Stack) dan FIFO (Queue).
5. **Searching**: Perbandingan efisiensi Linear Search vs Binary Search.
6. **Trees**: Struktur BST, aturan penempatan node, dan traversal.
7. **Hashing**: Fungsi Hash dan strategi menangani tabrakan data (Collision).

## Panduan Ujian Tengah Semester (Contoh)

Ujian biasanya terdiri dari:

- **Analisis Kode**: Mahasiswa diminta menentukan output dari potongan kode Linked List atau Stack yang diberikan.
- **Implementasi Fungsi**: Menulis fungsi C++ untuk operasi tertentu (misal: menghitung jumlah node dalam Tree).
- **Studi Kasus**: Memilih struktur data yang paling tepat untuk masalah tertentu (misal: fitur riwayat browser cocok menggunakan apa?).

## Kriteria Kesiapan

Mahasiswa dianggap siap jika mampu:

- Membuat simulasi manual (tracing) untuk operasi Insert/Delete pada Linked List.
- Menentukan Base Case pada fungsi rekursif.
- Menjelaskan perbedaan performa antara BST yang miring (skewed) dengan AVL Tree.

## Tips Persiapan

- Perbanyak latihan coding manual (tanpa IDE) untuk melatih logika pointer.
- Pahami visualisasi setiap struktur data dalam memori.
- Tinjau kembali tugas-tugas mandiri pertemuan sebelumnya.
