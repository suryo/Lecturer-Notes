## DESKRIPSI SINGKAT MK

Mata kuliah ini membahas berbagai tipe struktur data fundamental dalam ilmu komputer serta algoritma-algoritma yang terkait. Mahasiswa akan mempelajari cara merepresentasikan, mengorganisasi, dan memanipulasi data secara efisien menggunakan Array, Linked List, Stack, Queue, Tree, dan Graph. Selain itu, mahasiswa juga akan mempelajari algoritma pengurutan (Sorting) dan pencarian (Searching) beserta analisis kompleksitasnya menggunakan notasi Big-O. Perkuliahan menekankan kemampuan mahasiswa dalam memilih struktur data dan algoritma yang paling tepat untuk menyelesaikan masalah komputasi tertentu.

## BAHAN KAJIAN: MATERI PEMBELAJARAN

1. Pengantar Struktur Data & Analisis Algoritma (Big-O Notation)
2. Array (1D & 2D) — Operasi dasar, Matriks
3. Linked List — Singly, Doubly, Circular
4. Stack (Tumpukan) — Implementasi & Aplikasi
5. Queue (Antrian) — Linear Queue, Circular Queue, Priority Queue
6. Algoritma Sorting: Bubble Sort, Selection Sort, Insertion Sort
7. Algoritma Sorting Lanjutan: Merge Sort, Quick Sort
8. Algoritma Searching: Linear Search, Binary Search
9. Tree — Binary Tree, Traversal (Inorder, Preorder, Postorder)
10. Binary Search Tree (BST) — Insertion, Deletion, Search
11. Graph — Representasi (Adjacency Matrix & List)
12. Graph Traversal — BFS (Breadth-First Search), DFS (Depth-First Search)
13. Aplikasi Graph: Shortest Path (Dijkstra)
14. Proyek Akhir: Implementasi Struktur Data untuk Masalah Nyata
