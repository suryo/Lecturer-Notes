# Lab 13: Pengembangan Proyek Akhir

## Target Capaian

Mahasiswa menyelesaikan implementasi teknis proyek akhir mereka.

## Checklist Akhir

1. **Database Schema**: SQL (untuk data inti) & NoSQL (untuk data streaming/dokumen).
2. **Optimization**: Minimal ada 3 kueri yang sudah dioptimasi menggunakan Indexing/Stored Procedure.
3. **Security**: Bebas dari celah SQL Injection.
4. **Integration**: Database sudah terhubung ke aplikasi/dashboard.

## Instruksi Praktikum

Fokus pada debugging dan pembersihan kode. Gunakan AI (ChatGPT/DeepSeek) untuk membantu mencari bug logic atau menanyakan cara optimasi kueri yang buntu.
