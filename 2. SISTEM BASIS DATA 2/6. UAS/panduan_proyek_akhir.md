# Panduan UAS: Final Project - Hybrid Database Ecosystem

## Deskripsi Proyek

Membangun sistem informasi kompleks yang menggunakan pendekatan **Hybrid Database** (Relasional untuk data transaksi & NoSQL untuk data log/dokumen/IoT).

## Persyaratan (Requirements)

1. **Multi-Database**: Menggunakan minimal 1 SQL (MySQL/Postgres) dan 1 NoSQL (MongoDB/Firebase).
2. **Stored Logic**: Menggunakan SP dan Trigger untuk integritas data.
3. **Aggregation Dashboard**: Menampilkan laporan statistik yang diproses di sisi database.
4. **IoT/Real-time (Optional Valor)**: Integrasi dengan data sensor atau notifikasi real-time.
5. **Deployment Ready**: Database sudah diatur user privilege-nya.

## Rubrik Penilaian (40% dari Total Nilai Mata Kuliah)

- Arsitektur Database: 30%
- Optimasi & Performa: 30%
- Integrasi NoSQL: 25%
- Presentasi & Demo: 15%
