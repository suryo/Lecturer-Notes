# Panduan UTS: Performance Analysis Project

## Deskripsi

UTS berupa proyek analisis dan optimasi database relasional berskala menengah. Mahasiswa diminta menunjukkan kemampuannya dalam melakukan diagnosa dan perbaikan pada database.

## Tahapan Proyek

1. **Audit Database**: Melakukan `EXPLAIN ANALYZE` pada 10 kueri paling penting di sistem.
2. **Implementasi Index & SP**: Menerapkan perubahan teknis berdasarkan hasil audit.
3. **Security Check**: Memastikan login dan input data aman dari SQL Injection.
4. **Laporan Teknis**: Menyusun dokumentasi teknis yang membuktikan peningkatan performa sistem.

## Rubrik Penilaian

- Ketajaman Analisis: 30%
- Keberhasilan Optimasi: 40%
- Keamanan Data: 20%
- Kerapian Dokumentasi: 10%
