# Tugas 3: NoSQL Aggregation & Application Integration

## Deskripsi Tugas

Menghubungkan aplikasi ke database NoSQL (MongoDB/Firebase) dan melakukan pengolahan data agregasi untuk tujuan pelaporan.

## Kriteria Penilaian

1. **CRUD Document**: Berhasil melakukan operasi tambah/edit/hapus pada MongoDB.
2. **Aggregation Pipeline**: Membuat kueri laporan yang kompleks (misal: rata-rata rating per kategori) menggunakan pipeline `$match` dan `$group`.
3. **Web API**: Menyajikan hasil agregasi tersebut melalui REST API sederhana (PHP/Python).

## Instruksi

- Gunakan dataset `restaurants` di MongoDB.
- Buat aplikasi web sederhana (1 halaman) yang menampilkan data tersebut secara real-time.
