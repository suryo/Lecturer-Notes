# Kontrak Perkuliahan: Sistem Basis Data Lanjutan

## 1. Identitas Mata Kuliah

- **Nama Mata Kuliah**: Sistem Basis Data Lanjutan
- **Program Studi**: Teknik Informatika / Sistem Informasi
- **Dosen Pengampu**: Suryo Atmojo S.Kom M.Kom
- **Durasi**: 14 Pertemuan (1 Semester)
- **Metode**: Teori (40%) & Praktikum (60%)

## 2. Deskripsi Mata Kuliah

Mata kuliah ini membahas konsep lanjutan dalam sistem basis data, termasuk optimasi kueri, desain basis data yang kompleks, transaksi, replikasi, big data, serta integrasi dengan aplikasi web dan IoT. Mahasiswa akan memperoleh pemahaman mendalam mengenai manajemen basis data relasional dan non-relasional serta penerapan teknologi basis data dalam skenario dunia nyata.

## 3. Platform & Tools

- **DBMS**: MySQL, PostgreSQL, MongoDB, Firebase, SQL Server
- **Bahasa**: SQL, PL/SQL, Python, PHP
- **AI Support**: ChatGPT, BlackBoxAI, DeepSeek

## 4. Materi Pembelajaran (14 Pertemuan)

1.  **Pengenalan Basis Data Lanjutan & Optimasi SQL**: Review SQL & Indexing Dasar.
2.  **Indexing & Query Performance Tuning**: Jenis Indexing & Case Study Dataset Besar.
3.  **Stored Procedures & Functions**: Implementasi di MySQL & PostgreSQL.
4.  **Triggers & Event Scheduling**: Automasi database & Event Scheduler.
5.  **Transaksi & Manajemen Konkurensi**: Properti ACID & Isolation Levels.
6.  **Replikasi & Backup Database**: Master-Slave, Master-Master, & PITR.
7.  **Pengenalan NoSQL (MongoDB, Firebase)**: SQL vs NoSQL & CRUD Dasar.
8.  **Struktur Data NoSQL & Agregasi**: Model Data NoSQL & Aggregation Framework.
9.  **Big Data & Data Warehousing**: OLTP vs OLAP, BigQuery, & Redshift.
10. **Integrasi Web**: REST API & Database Connection (PHP/Python).
11. **Basis Data untuk IoT**: Firestore SDK & Sensor Data Management.
12. **Keamanan Basis Data**: SQL Injection Prevention & NoSQL Security.
13. **Pengembangan Proyek Akhir**: Debugging & Performance Testing.
14. **Presentasi & Evaluasi Proyek**: Demo & Review hasil akhir.

## 5. Komponen Penilaian

- **Tugas & Praktikum**: 30%
- **Proyek Akhir (UAS Project)**: 40%
- **Ujian Akhir (Teori/UAS Exam)**: 30%

## 6. Referensi

- Silberschatz, A., Korth, H. F., & Sudarshan, S. (2019). _Database System Concepts_.
- Dokumentasi Resmi MySQL, PostgreSQL, MongoDB, & Firebase.
