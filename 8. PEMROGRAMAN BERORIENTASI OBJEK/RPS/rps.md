## DESKRIPSI SINGKAT MK

Mata kuliah ini membahas paradigma Pemrograman Berorientasi Objek (OOP) secara teoritis dan praktis menggunakan bahasa pemrograman Java. Mahasiswa akan mempelajari konsep dasar OOP mulai dari Class dan Object, hingga empat pilar utama yaitu Encapsulation, Inheritance, Polymorphism, dan Abstraction. Selanjutnya mahasiswa akan diperkenalkan dengan Interface, Design Patterns, penanganan Exception, penggunaan Collections Framework, File I/O, dan pengembangan antarmuka pengguna (GUI) sederhana. Perkuliahan diorientasikan pada kemampuan mahasiswa membangun aplikasi terstruktur, modular, dan mudah dipelihara menggunakan prinsip-prinsip OOP.

## BAHAN KAJIAN: MATERI PEMBELAJARAN

1. Paradigma OOP vs Prosedural — Konsep dan Keunggulan
2. Class, Object, Atribut, Method, dan Constructor
3. Encapsulation — Access Modifier, Getter & Setter
4. Inheritance — `extends`, `super`, Method Overriding
5. Polymorphism — Overloading vs Overriding, Upcasting & Downcasting
6. Abstraction — Abstract Class & Interface
7. Design Patterns — Singleton, Factory Method, Observer
8. Exception Handling — `try-catch-finally`, Custom Exception
9. Collections Framework — `ArrayList`, `LinkedList`, `HashMap`, `HashSet`
10. File I/O — `FileReader`, `FileWriter`, `BufferedReader`, Serialisasi Objek
11. Java Generics & Lambda Expression
12. GUI Dasar dengan Java Swing / JavaFX
13. Event Handling pada GUI
14. Proyek Akhir: Aplikasi Desktop Berbasis OOP
