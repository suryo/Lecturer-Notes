# Panduan UAS: Proyek Akhir Smart Solution Ecosystem

## Deskripsi Proyek

Membangun solusi sistem tertanam yang menyelesaikan masalah spesifik di dunia nyata menggunakan kombinasi Arduino/ESP32 dan Raspberry Pi dengan integrasi Cloud.

## Tema Pilihan (Pilih Salah Satu)

1.  **Smart Agriculture**: Monitoring kelembapan tanah & penyiraman otomatis via Web.
2.  **Smart Security**: Deteksi orang dengan kamera Raspberry Pi & Notifikasi Cloud.
3.  **Industrial Automation**: Kendali motor/actuator jarak jauh dengan dashboard data logger.
4.  **Health Monitoring (Wearable Concept)**: Pengukuran suhu/detak jantung (simulasi) & Penyimpanan database Cloud.

## Komponen Wajib

1.  **Multi-Platform**: Integrasi Arduino/ESP32 untuk akuisisi data dan Raspberry Pi untuk pemrosesan/gateaway data.
2.  **Connectivity**: Penggunaan protokol komunikasi (MQTT/HTTP/I2C) antar perangkat.
3.  **Storage**: Penyimpanan data ke database (MySQL/Firebase).
4.  **Presentation**: Demonstrasi sistem yang berfungsi (Live Demo).

## Kriteria Kelulusan (40% dari Total Nilai Mata Kuliah)

- Fungsionalitas Sistem (Apakah sistem bekerja sesuai rencana?): 40%
- Arsitektur & Interkoneksi (Cara perangkat berkomunikasi): 30%
- Inovasi & Kebermanfaatan: 20%
- Presentasi & Tanya Jawab: 10%
