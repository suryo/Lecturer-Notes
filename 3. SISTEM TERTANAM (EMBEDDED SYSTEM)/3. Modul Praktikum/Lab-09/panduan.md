# Lab 9: Database Log dan Firebase

## Tantangan Mandiri (Contoh)

### Tantangan 1: Firebase Latch

Ganti variabel di console, LED merespon.

> [!IMPORTANT] **Solusi Tantangan**
> ESP32 melakukan `Firebase.RTDB.getBool` secara looping atau menggunakan listener stream.

### Tantangan 2: History Export

Konversi log Firebase ke CSV.

> [!IMPORTANT] **Solusi Tantangan**
> Export database dari menu Firebase Console (.json), lalu buka di Excel (Data > Get Data > from JSON).
