# Lab 8: UTS - Project Finalization

## Target Capaian

Mahasiswa mampu menyatukan seluruh materi Lab 1-7 menjadi satu aplikasi **Company Profile** yang utuh.

## Checklist Persiapan UTS

1. **Model & Migration**: Pastikan tabel profiles sudah ada datanya.
2. **API Controller**: Pastikan method GET, POST, dan Upload Image berfungsi.
3. **Flutter App**:
   - Layout menggunakan Master/Main Layout.
   - Mengambil data profil dari API.
   - Form untuk update profil dan upload logo.

## Langkah Praktikum

Ingatkan kembali mahasiswa untuk mencoba aplikasi di device asli (jika memungkinkan) atau mengatur IP Address `10.0.2.2` jika menggunakan emulator Android agar bisa terhubung ke `localhost` komputer.

## Tugas Review

Bersihkan kode (Refactoring), pisahkan widget menjadi file terpisah untuk memudahkan maintenance.
