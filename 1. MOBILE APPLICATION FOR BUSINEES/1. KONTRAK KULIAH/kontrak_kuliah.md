# Kontrak Perkuliahan: Mobile Application for Business

## 1. Identitas Mata Kuliah

- **Nama Mata Kuliah**: Mobile Application for Business
- **Teknologi Backend**: Laravel 10 (Web Admin & RESTful API)
- **Teknologi Mobile**: Flutter Framework
- **Prasyarat**: Pemrograman Web Dasar, Dasar Pemrograman Mobile.

## 2. Deskripsi Mata Kuliah

Mata kuliah ini membekali mahasiswa dengan keahlian membangun solusi digital untuk kebutuhan bisnis modern. Fokus utama adalah integrasi antara sistem backend (Laravel) sebagai pusat data dan aplikasi mobile (Flutter) sebagai antarmuka pengguna, guna menciptakan ekosistem aplikasi bisnis yang efisien dan skalabel.

## 3. Capaian Pembelajaran (CPMK)

1. Mahasiswa mampu merancang arsitektur aplikasi bisnis yang menghubungkan API Laravel dengan Frontend Flutter.
2. Mahasiswa mampu membangun fitur-fitur utama bisnis seperti manajemen produk, autentikasi, dan pelaporan.
3. Mahasiswa mampu mengimplementasikan studi kasus nyata dalam proses bisnis.
4. Mahasiswa mampu mendemonstrasikan aplikasi mobile business yang fungsional dan siap pakai.

## 4. Materi Pembelajaran (14 Pertemuan)

1.  **Intro to Mobile Business Apps**: Arsitektur Laravel-Flutter & Setup Environment.
2.  **Flutter UI Fundamentals**: Layouting untuk Aplikasi Bisnis (Stateless & Stateful).
3.  **Advanced UI & State Management**: Interaktivitas & Manajemen State (Provider/Bloc).
4.  **Backend Foundation**: Dasar Laravel 10, Migrations, & RESTful API Dasar.
5.  **Studi Kasus 1 - Company Profile**: Integrasi API Laravel ke Flutter (Read/Get Data).
6.  **Multimedia Management**: Handling Image Upload (Laravel Storage) & Flutter Image Picker.
7.  **Data Interaction**: Handling Form Submission & Feedback (Post Data).
8.  **UTS - Midterm Project**: Penyelesaian Aplikasi Company Profile.
9.  **Studi Kasus 2 - Online Shop (P1)**: Database Relasional Kompleks & Sanctum Auth API.
10. **Authentication Flow**: Login, Register & Token Persistence di Flutter.
11. **E-Commerce Features**: Katalog Produk, Searching, & Filtering Logic.
12. **Transaction Logic**: Implementasi Shopping Cart & Checkout API.
13. **Real-time Notifications**: Push Notification & Order Tracking Concept.
14. **Deployment**: Build APK/Bundle & Hosting Laravel API (Production Readiness).

## 5. Tugas Mandiri & Kelompok

Selain studi kasus utama, mahasiswa akan diberikan tugas untuk membangun sistem bisnis lainnya, meliputi:

- **Sistem Pengelolaan Karyawan**: Fitur presensi (absen) dan jurnal pekerjaan harian.
- **Sistem Pengelolaan Aset Perusahaan**: Pencatatan dan monitoring aset fisik.
- **Sistem Pengelolaan Gudang (Warehouse Management)**: Manajemen stok dan arus barang masuk-keluar.

## 6. Komponen Penilaian

- **Kehadiran**: 10%
- **Tugas Praktikum (Employee/Asset/Warehouse System)**: 25%
- **UTS (Company Profile Project)**: 30%
- **UAS (Online Shop Final Project)**: 35%

## 7. Referensi & Tooling

- **IDE**: Visual Studio Code / Android Studio
- **Backend Server**: XAMPP / Laragon (PHP 8.1+, MySQL)
- **Framework**: Laravel 10, Flutter 3.x
- **API Testing**: Postman / Insomnia
