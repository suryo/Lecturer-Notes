## DESKRIPSI SINGKAT MK

Mata kuliah ini membahas konsep dan praktik pengembangan aplikasi mobile dan multi-platform yang difokuskan pada kebutuhan bisnis modern. Mahasiswa akan mempelajari proses analisis kebutuhan bisnis, perancangan arsitektur sistem, implementasi backend (API/Cloud), dan pengembangan antarmuka pengguna (UI/UX) menggunakan framework lintas platform seperti Flutter. Mahasiswa juga akan dilatih melakukan evaluasi kualitas aplikasi melalui pengujian fungsional dan usability.

---

## BAHAN KAJIAN: MATERI PEMBELAJARAN

Materi pembelajaran dijabarkan dalam pokok-pokok bahasan sesuai Sub-CPMK:
1. Pengantar pengembangan aplikasi bisnis multi-platform
2. Analisis kebutuhan dan perancangan sistem (ERD, UML, Wireframe)
3. Pengenalan dan setup Flutter & Dart
4. Manajemen state dan navigasi dalam Flutter
5. Desain UI/UX yang berorientasi pengguna (Figma → Flutter)
6. Konsumsi REST API (Laravel Backend) dari Flutter
7. Autentikasi dan Otorisasi (JWT/Firebase Auth)
8. Integrasi penyimpanan data (SQLite lokal & Cloud Firebase)
9. Pengujian fungsional dan UI Testing
10. Deployment aplikasi (Google Play Store / PWA)
11. Evaluasi usability dengan metode heuristik
