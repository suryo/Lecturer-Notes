# Panduan UAS: Final Project - Online Shop Ecosystem

## Tema Proyek

Membangun ekosistem e-commerce lengkap (Frontend Mobile & Backend API) dengan fitur transaksi utuh.

## Fitur Wajib (Requirements)

1.  **Authentication**: Register & Login menggunakan Laravel Sanctum Token.
2.  **Session Persistence**: User tidak perlu login ulang saat aplikasi ditutup (Shared Preferences).
3.  **Katalog Produk**: List barang dengan fitur search & filter kategori.
4.  **Manajemen Keranjang (Cart)**: Menambah, menghapus, dan menghitung total belanja secara lokal.
5.  **Checkout & Transaction**: Mengirim data keranjang ke server dan mendapatkan nomor invoice.
6.  **Multimedia**: Setiap produk wajib memiliki gambar.

## Tambahan (Nilai Plus)

- Implementasi Notifikasi (FCM).
- Fitur Dark Mode.
- Fitur History Transaksi.

## Rubrik Penilaian

- **Keamanan (Auth & Token)**: 25%
- **Manajemen State (Cart Logic)**: 25%
- **Integritas Transaksi (Checkout)**: 25%
- **Estetika & UX**: 15%
- **Dokumentasi API**: 10% (README yang menjelaskan endpoint).
