# Tugas 1: Employee Management System (Absensi & Jurnal)

## Deskripsi Tugas

Membangun sebuah aplikasi mobile sederhana untuk manajemen karyawan. Fokus utama tugas ini adalah pengolahan form, validasi input, dan pengiriman data ke backend.

## Kriteria Penilaian

1.  **Form Input (Flutter)**: Membuat form untuk absensi (Nama, Posisi, Jam) dan jurnal harian (Aktivitas, Durasi).
2.  **Validasi (Flutter)**: Semua field wajib diisi dan memiliki format yang benar.
3.  **Integrasi Backend (Laravel)**: Menyimpan data ke database melalui API.
4.  **Feedback User**: Menampilkan konfirmasi (SnackBar/Toast) setelah data berhasil dikirim.

## Instruksi Pengerjaan

1.  Buatlah tabel `attendance` dan `work_logs` via Migrations.
2.  Desain UI di Flutter menggunakan `TextFormField` dan `Form` widget.
3.  Implementasikan pengiriman data menggunakan method `POST` ke API Laravel.
4.  Pastikan data tersimpan secara permanen di database MySQL.
